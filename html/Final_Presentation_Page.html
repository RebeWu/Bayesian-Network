A Bayesian network is a probabilistic graphical model (a type of statistical model) that represents a set of random variables and their conditional dependencies via a directed acyclic graph (DAG).

Here are a series of simple examples to show what Bayesian Networks look like and how to use them to prodict the answer you want. The Bayes Networks are really useful (such as the bayes rnn)— but only if you know how to build one.

Before diving in: if you haven’t encountered Bayes algorithms before, here’s what you need to know about the math behind it. Formally, Bayesian networks are DAGs whose nodes represent random variables in the Bayesian sense: they may be observable quantities, latent variables, unknown parameters or hypotheses. Edges represent conditional dependencies; nodes that are not connected represent variables that are conditionally independent of each other. Each node is associated with a probability function that takes, as input, a particular set of values for the node's parent variables, and gives (as output) the probability of the variable represented by the node.

Besides the graph, the Bayesian Networks include a Conditional Probability Table (CPT) which defined for a set of discrete (not independent) random variables to demonstrate marginal probability of a single variable with respect to the others. Here are some math fundamentals:

Conditional Probability : P(B|A) = P(B,A)/P(A)

Product Rule : P(B,A) = P(B|A)·P(A)

If two nodes are marginal independent , it means: P(A|B) =P(A) ,P(B|A)=P(B). If three nodes are conditional independent ,such as A and B are independent but C is not, it means P(A|B,C) = P(A|C)

If we can know all different node sets' joint distribution, we can solve all the statisc problems, but the truth is it is too difficult to calculate all the probabilities with a large number of nodes.We need find some ways to simplify the question.

An animation demonstrating the EM algorithm fitting a two component Gaussian mixture model to the Old Faithful dataset. The algorithm steps through from a random initialization to convergence.

A more fully Bayesian approach to parameters is to treat parameters as additional unobserved variables and to compute a full posterior distribution over all nodes conditional upon observed data, then to integrate out the parameters. This approach can be expensive and lead to large dimension models, so in practice classical parameter-setting approaches are more common.

4. A Hard Problem ———— Structure learning

In the simplest case, a Bayesian network is specified by an expert and is then used to perform inference. In other applications the task of defining the network is too complex for humans.

Actually the structure learning of bayesian network is a NP-hard problem,but we have no choice and the network structure and the parameters of the local distributions must be learned from data.
